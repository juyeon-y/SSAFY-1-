# 비밀번호 찾기

- 로그인창에서 'Forgot Password?'를 누르게 되면 비밀번호 찾기 페이지로 이동합니다.
- 비밀번호 찾기는 아이디와 비밀번호를 입력하도록 되어있습니다.
- 추후 사용자 정보를 확인하여 있는 회원일 경우 이메일로 임시 비밀번호를 발급하는 방식을 생각하고 있습니다.
